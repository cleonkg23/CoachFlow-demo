# CoachFlow — Interactive Portfolio Demo

CoachFlow is an AI-assisted intervention management tool designed to help apprenticeship coaches identify learners requiring attention and take consistent, evidence-based action.

## Demo

## What the demo shows

- Prioritised learner intervention queue
- Explainable risk signals
- Recommended next actions
- Context-aware communication drafts
- Human review and approval
- Intervention history and status tracking

## About this version

This is a portfolio-safe frontend simulation built with fictional data.

The original application integrated with Google Sheets, Gmail, Google Calendar, Google Tasks and an AI model. Those integrations have been intentionally removed from this public demonstration.

No real learner or employer data is included.

## Running locally

Open `index.html` through a local development server, or embed the supplied web component in another webpage.
